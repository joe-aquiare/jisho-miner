# Icons

Place three PNG icon files here:

- `icon16.png` — 16×16 px
- `icon48.png` — 48×48 px
- `icon128.png` — 128×128 px

These are referenced by `manifest.json` for the extension icon and toolbar button.
Until real icons are added the extension will load without them but Chrome will show a
broken-image placeholder in the toolbar.
